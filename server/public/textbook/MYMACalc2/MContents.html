In these contents, the Techniques of Integration are all listed before the Applications of Integration. If you prefer, it is OK to alternate between Techniques and Applications, provided when you read a particular Application, you already know the Techniques which are listed in this contents. On the other hand, if you cover all the Techniques first, then you will need to postpone any exercise on an Application until after you have covered that Application.
